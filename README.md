## Overview

Base de datos sobre alfabetización financiera en Iberoamérica (2000-2026). Este dataset recopila datos sobre niveles de educación financiera, competencias económicas, programas de formación financiera y políticas públicas de inclusión financiera en países iberoamericanos.
